SYNOPSIS

function [APSF] = deconSTORM_prepareAPSF(sigma,Npixels,npixels)

DESCRIPTION 

 [APSF] = deconSTORM_prepareAPSF(sigma,Npixels,npixels)

 Create a Gaussian point spread function (PSF) convolutional transfer
 function matrix

 Inputs:
 sigma - Standard deviation parameter of the Gaussian PSF shape, in pixels
 Npixels - linear dimension, in pixels, of the image field of view
           (assumed square)
 npixels - Linear dimension (in pixels) of the super-resolution estimate

 Output:
 APSF - transfer function mapping from the high- to the 
 low-resolution images, of size (npixels^2) x (Npixels^2)
